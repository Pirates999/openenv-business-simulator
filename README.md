# 🚀 OpenEnv – AI Business Simulator

## 📌 Overview
This project is an AI environment where an agent learns how to run a business using decision-making.

---

## ⚙️ Functions

### reset()
Resets the environment

### state()
Shows current condition:
- money
- employees
- revenue

### step(action)
Takes action and updates environment

---

## 🎮 Actions
- hire → increases employees
- invest → increases revenue
- save → collects money

---

## 🧠 Reward
Reward is based on:
- money
- revenue

---

## ▶️ How to Run

Open terminal and type:

python main.py

---

## 🏆 Project Goal
To simulate real-world business decision-making for AI training.
